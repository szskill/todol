# TodoL

## A language for TODOs.

### Example syntax

```todol
[x] Make cake
[x] Decorate house
[x] Get balloons
[ ] Wait for people to come
```

### Running

```sh
# Parse a TodoL file into JSON
todol

# Parse a TodoL file into JSON with a different filename
todol ANOTHER_TODO

# Supported languages:
#   [x] JSON
```
